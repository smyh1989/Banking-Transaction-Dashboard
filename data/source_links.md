# Data Sources

## Primary Dataset
**Financial Transactions Dataset**
- Source: Kaggle
- Link: https://www.kaggle.com/datasets/computingvictor/transactions-fraud-datasets
- Files used:
  - transactions_data.csv (1.2 GB, 13M+ rows)
  - cards_data.csv (498 MB)
  - users_data.csv (161 KB)
  - mcc_codes.json (5 KB)

## Note
Due to file size limitations, raw data files are not included in this repository.
Please download directly from Kaggle using the link above.
